[DeGov Square](https://square.degov.ai) is an on‑chain SaaS governance platform built on the latest [OpenZeppelin Governor](https://docs.openzeppelin.com/contracts/5.x/governance). It leverages the open‑source [degov](https://github.com/ringecosystem/degov) core UI to provide a simple, powerful interface for interacting with Governor-based DAOs.
