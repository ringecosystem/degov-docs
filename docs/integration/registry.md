# DAOs Registry

The [DeGov Registry](https://github.com/ringecosystem/degov-registry) lists DAO configuration files for DeGov-powered deployments. These `degov.yml` files are the reference source for how real DAOs are configured.

Use the registry when you want to:

- Review existing DAO configuration examples
- Compare chain IDs, governor addresses, and other settings
- Model your own `degov.yml` after a working deployment

The DAOs shown on [DeGov Square](https://square.degov.ai) are sourced from this registry, so it is the canonical place to inspect live DAO configs.

You can also open a DAO's config directly from its dashboard. For example, clicking the edit button:

![DAO configuration link on dashboard](../images/registry-1.png)

This opens the corresponding `degov.yml` file in the registry, such as [ring-dao.yml](https://github.com/ringecosystem/degov-registry/blob/main/daos/ring-dao.yml).
